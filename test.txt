Общая структура
Код представляет собой модуль, который, используется для работы с базами данных 
и их структурой в формате XML. Он включает в себя несколько функций, 
каждая из которых выполняет определенные задачи.

Основные функции

CleanStr(Str)
Эта функция очищает строку от небуквенных символов и лишних пробелов.
Удаляет пробелы в начале и конце строки, а также заменяет последовательные пробелы на один.
Удаляет символы с определенными кодами ASCII, которые указаны в таблице ASCII_SymbolsCode.

CleanXmlTagName(Str)
Эта функция очищает имя XML-тэга от запрещенных символов.
Использует CleanStr для предварительной очистки строки.
Заменяет пробелы на подчеркивания, удаляет недопустимые символы и приводит строку к верхнему регистру.

DatabaseStructureToXML(DataBase)
Эта функция принимает экземпляр базы данных и возвращает XML-структуру, описывающую ее.
Создает корневой узел XML с именем базы данных.
Устанавливает атрибуты для базы данных, такие как db_mnemo и db_number.
Проходит по полям базы данных и создает узлы для каждого поля, устанавливая соответствующие атрибуты.
Если у поля есть связанные базы, создает узлы для них.

CheckMnemoCode(Bases)
Эта функция проверяет уникальность мнемокодов для баз данных.
Создает таблицу BasesTable, в которой хранятся коды баз.
Если мнемокод содержит символ $, функция пытается создать уникальный код, добавляя буквы из алфавита, пока не найдет уникальный код.
Если не удается найти уникальный код, выводит сообщение об ошибке.
